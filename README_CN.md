# 修改说明（仅列出本地改动）

1. **保留打包与非打包两套运行逻辑：** 在可执行文件旁边创建 `Logs/` 与 `Config/` 目录保存日志和设置，同时自动检测运行环境，未打包时不再捆绑 Windows App Runtime。  
2. **未充电持续提醒加强：** 拔掉电源后，10 秒轮询计时器会按设定的分钟间隔触发提醒气泡，包含「我知道了」和「延后提醒」按钮（后者支持下拉选项，选择自动持久保存）；未点击「我知道了」前计时不会重置，插上电源后计时器立即停止，避免不必要的唤醒。此功能用于提醒用户插电使用电脑，延长电池寿命，减缓电池损耗。  
3. **导航不再无限压栈：** 在设置和关于之间切换默认清空回退栈，只有从关于返回设置时才会让返回箭头亮起；反复点击不会堆栈。  
4. **托盘图标和开机自启：** 非商店运行时标题栏图标由本地 `Assets/logo.ico` 加载，默认不开启“开机自启”，必须手动切换。
